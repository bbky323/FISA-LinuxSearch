# 🔎FISA-LinuxSearch
이번 학습에서는 Linux 환경에서 자주 사용하는 파일 검색, 문자열 검색, 텍스트 가공, 자동화 스크립트 작성, 그리고 JSON/YAML 검수 도구를 익힌다.
단순히 명령어 문법만 보는 것이 아니라, 로그 분석, 설정 파일 검수, 파일 탐색, 데이터 추출 및 집계처럼 실제 운영 환경에서 자주 발생할 수 있는 상황을 가정하여 문제를 해결하는 방식으로 학습한다.

주요 목표는 다음과 같다.
- 원하는 파일을 조건에 따라 정확히 찾기
- 여러 파일에서 특정 문자열이나 패턴을 검색하기
- 로그나 텍스트 데이터를 행/열 단위로 분석하기
- 반복 작업을 쉘 스크립트로 자동화하기
- JSON, YAML 형식의 데이터와 설정 파일을 검수하기
- 핵심 키워드: **find, grep, awk, jq, yq, crontab, Shell script, Shell**
